## Маршрутизация ячеек в EME.WMS

в системе EME.WMS маршрутизация ячеек определяет порядок и направление движения товаров по складским зонам и секциям.

### Основные понятия

- **Ячейка** — минимальная адресуемая единица хранения на складе
- **Зона** — логическая группировка ячеек
- **Секция** — физическая область склада
- **Маршрут** — последовательность ячеек для выполнения складской операции

### Применение маршрутизации

Маршрутизация ячеек используется при:

- Размещении товара (определение оптимальной ячейки)
- Подборке заказа (оптимизация пути кладовщика)
- Перемещении товара между зонами
- Инвентаризации (определение порядка обхода)
