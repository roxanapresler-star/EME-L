## Подключение считывателя отпечатков в ЕМЕ БД

В ЕМЕ БД имеется возможность использовать в диалогах устройства с последовательным интерфейсом, включая считыватель отпечатков. Устройство может подключаться к COM-порту или по USB.

в системе EME.WMS для подключения внешнего устройства необходимо задать параметры в настройках рабочей станции (вкладка "Последовательные интерфейсы"): id, драйвер, порт устройства.

### Настройка обработчика ввода

```EME-L
'В конструкторе класса:'
Dialog = Object("Dialog");

'При загрузке диалога:'
Dialog_OnLoad()
{
    Dialog.LoadScannerDriver(0); '0 — id устройства'
}

'При закрытии диалога:'
Dialog_OnClose()
{
    Dialog.UnloadScannerDriver(0);
}
```

### Флаг "Проверять отпечаток"

В настройках диалога имеется флаг "Проверять отпечаток". В текущей версии EME.WMS данная настройка не используется.
