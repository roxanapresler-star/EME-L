# Развертывание релиз-версии ЕМЕ БД

> **Синонимы для поиска:** релиз EME БД, установка EME WMS, развертывание WMS, WMS-Release, автоматическая установка, ручная установка EME, релиз-версия базы данных, инсталляция EME.WMS, профессиональная версия, стандартная версия, демо-версия, eme.config, настройка ярлыка db.exe

---

## Общие сведения

Релиз-версия ЕМЕ БД — текущая и стабильная на данный момент времени сборка базы данных, пригодная для промышленной эксплуатации.

в системе EME.WMS актуальный релиз хранится на сетевом диске **«Exchange»** по адресу `I:\WMS-Release`. Внутри находятся две директории:

- **EME.WMS Release — автоматическая установка**
- **EME.WMS Release — ручная установка**

### Варианты автоматической установки

При автоматической установке пользователь выбирает один из вариантов:

- **Профессиональная версия** — полный набор модулей
- **Стандартная версия** — базовый набор модулей
- **Демо-версия**
- **Версия только с рабочей станцией**

### Ручная установка

в EME БД при ручной установке внутри папки «Данные» имеется архив `db_wms_release`. Его нужно разархивировать в рабочую директорию ЕМЕ БД. **В пути и названии папки не должно быть пробелов** — иначе программа будет выдавать ошибки при загрузке срезов.

Также из директории `WMS-Release` нужно скопировать приложение **«TaskMonitor»** (вместе с исходниками тем оформления) в директорию внутри рабочего каталога ЕМЕ БД.

в системе EME.WMS для успешного запуска нужно исправить адреса в ярлыке `db.exe`: правая кнопка мыши → «Свойства» → заменить пути в строках «Объект» и «Рабочая папка» в соответствии с рабочим каталогом ЕМЕ БД (указывается полный путь).

## Пошаговая инструкция: автоматическая установка

1. Открыть установщик EME.WMS и следовать его инструкциям
2. Перейти в директорию с установленной EME.WMS и выбрать по названию ярлыка требуемый режим запуска (например: `HttpMono`)
3. Выполнить настройку сетевых параметров в рабочей станции EME БД для корректной работы Монитора Заданий и ТСД
4. Выполнить развертывание Хэш-БД
5. Выполнить настройку ТСД для работы с http-сервером EME БД
6. Запустить `.exe`/`.jar` файл Монитора Заданий из директории `TaskMonitor`, ввести корректный IP-адрес с портом запущенной EME.WMS и войти под сотрудником с должностью **«Оператор БД»**

---

## Пошаговая инструкция: ручная установка

1. Установить Java для работы с Монитором Заданий (`https://www.java.com/en/download/`)
2. Настроить адреса запуска программы в ярлыке `db`, расположенном в корне рабочей директории ЕМЕ БД
3. Настроить требуемые параметры запуска ЕМЕ БД в файле `eme.config`: указать режим запуска `/mono` и дополнительные ключи `/http` и `/bc`
4. Запустить базу данных с указанными параметрами, используя ярлык `db.exe` в корне рабочего каталога
5. Выполнить настройку сетевых параметров в рабочей станции EME БД для корректной работы Монитора Заданий и ТСД
6. Выполнить развертывание Хэш-БД
7. Выполнить настройку ТСД для работы с http-сервером EME БД
8. Запустить `.exe`/`.jar` файл Монитора Заданий, ввести корректный IP-адрес с портом запущенной EME.WMS и войти под сотрудником с должностью **«Оператор БД»**
