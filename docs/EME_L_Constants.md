## 1. Логические константы и константы записей/полей в ЕМЕ БД

в языке EME-L используются следующие логические константы:

| Константа | Значение | Описание |
|-----------|----------|----------|
| `TRUE` | -1 | Истинное значение (может использоваться для изменения поведения метода по умолчанию) |
| `FALSE` | 0 | Ложь (возвращаемое значение по умолчанию в EME-L методах) |
| `EMPTY` | — | Пустое значение |

Для определения значений записей/полей в БД и их проверки на валидность:

| Константа | Значение | Описание |
|-----------|----------|----------|
| `NULL_REF` | -1 | NULL-ссылка на строку записи БД |
| `NEW_LINE_MARK` | 2000000000 | Маркер новой строки (ещё не заполненной) |
| `NULL_RECORD` | -100 | Несуществующая запись в БД |
| `NULL_FIELD` | -100 | Несуществующее поле в записи БД |
