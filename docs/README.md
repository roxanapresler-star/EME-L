## Как пользоваться документацией

### Для программистов

1. Используйте русский язык для поиска (например: "как создать транзакцию", "обход строк документа")
2. Код в примерах — на английском (EME-L синтаксис)
3. Комментарии в коде — на русском для понимания логики

### Примеры поисковых запросов

- "как обрезать строку в EME-L"
- "фильтрация записей в SkipMode"
- "создание диалога в EME.WMS"
- "обработчик ончек валидация"
- "SQL запрос GROUP BY пример"
- "режимы запуска EME БД"

## Стандарты документации

Документация следует стандартам EME-L для Context7:

1. **Русское описание** — для семантического поиска
2. **Английский код** — синтаксис EME-L остаётся на английском
3. **Семантические якоря** — множественные синонимы для каждого метода
4. **Контекстные якоря** — упоминания "в EME-L", "в системе EME.WMS" каждые 2-3 абзаца
5. **Few-Shot примеры** — полные рабочие примеры с русскими комментариями
