> **Поисковые синонимы:** Linux-сервер, Astra Linux, EME.WMS Linux, консоль сервера, редактор ресурсов, Монитор Заданий, интерактивный интерфейс, парадигма Linux, диагностика Linux, сервер WMS Linux, отсутствие рабочей станции, человек-машина

---

## Linux-сервер в ЕМЕ

В системе Linux, в отличие от Windows, **отсутствует графический интерфейс** рабочей станции EME.WMS. Вместо этого применяется парадигма полного переноса пользовательского функционала в **Монитор Заданий** (создание документов прихода, просчёт пачки заказов, распределение приказов по кладовщикам и т.д.).

Сервер WMS на Linux выступает в роли **генератора складских заданий** и связующего звена для клиентских приложений: ТСД, Монитор Заданий и других.

Отказ от рабочей станции позволяет полностью перевести систему EME.WMS в интерактивный интерфейс «человек-машина» в экосистеме Linux.

## Клиентские приложения для диагностики

С целью диагностики работы EME.WMS на Linux ведётся разработка двух клиентских приложений:

### Консоль сервера

Функционал аналогичен вкладке «Консоль сервера» в рабочей станции EME.WMS. Позволяет:

- Просматривать **статистику транзакций** (продолжительность, время ожидания)
- Отслеживать **текущие подключения** к серверу (параметры, ключи запуска)
- Просматривать и изменять **настройки рабочей станции**

### Редактор ресурсов

Функционал аналогичен вкладке «Ресурсы» в рабочей станции EME.WMS. Позволяет:

- Работать с **редактором классов** EME-L
- Работать с **редактором структуры базы данных**
- Работать с **редактором запросов**
- Производить **отладку EME-L кода**
- Настраивать **структуру базы данных** в системе EME.WMS на Linux
