## Создание локальной тестовой среды ЕМЕ БД

Локальная тестовая среда ЕМЕ БД позволяет осуществлять проверку выполненных доработок программы (тестирование срезов, отладка EME-L кода и др.), минимизируя риски для продуктивной среды.

в системе EME.WMS в локальной тестовой среде используются следующие компоненты:

- База данных, запущенная в однопользовательском режиме (**моно-режим**)
- Монитор Заданий (десктопный)
- ТСД (физическое устройство)
- Приложение для ТСД под Android (`.apk`-файл)

### Исходник приложения ТСД

в EME БД исходник приложения для ТСД под Android расположен по адресу `I:\WMS-Release`. Нужно зайти в папку с автоматической/ручной установкой, выбрать папку **«Терминал»** и выкачать `.apk` файл с приставкой **`arm`** в названии.

Если требуется полноценная среда для тестирования — её организация зависит от внутренних регламентов предприятия и средств управления проектами.

### Пошаговая инструкция создания локальной тестовой среды

1. Настроить параметры запуска ЕМЕ БД в файле `eme.config`: режим `/mono` + ключи `/http` и `/bc`
2. Запустить базу данных с указанными параметрами, используя ярлык `db.exe` в корне рабочего каталога
3. Настроить сетевые параметры рабочей станции для взаимодействия с Монитором Заданий и ТСД через диалог **«Настройки рабочей станции»**
4. Запустить `.jar`/`.exe` файл Монитора Заданий, ввести корректный IP-адрес с портом запущенной EME.WMS и войти под сотрудником с должностью **«Оператор БД»**
5. Подключить ТСД к компьютеру, загрузить и установить `.apk` файл приложения, настроить подключение к серверу EME.WMS и авторизоваться

## Настройка ТСД для работы с EME.WMS

### Шаг 1: Подключение ТСД и передача файла

в системе EME.WMS первым делом необходимо подключить ТСД к компьютеру через USB-шнур. В настройках USB-подключения выбрать **«Передача файлов»**.

На компьютере в проводнике выбрать наименование ТСД, перейти в директорию **`Download`** и скопировать туда `.apk` файл приложения.

### Шаг 2: Установка приложения

На ТСД через файловый менеджер перейти в директорию загрузок и нажать на `.apk` файл — установка начнётся автоматически.

### Шаг 3: Настройка сетевых параметров ТСД

После появления ярлыка приложения на главном экране ТСД:

1. Открыть приложение
2. Нажать кнопку **«Меню»** (правый верхний угол)
3. Перейти на вкладку **«Настройки»**
4. Указать корректный **IP-адрес с портом** запущенной EME.WMS

При успешной настройке приложение предложит произвести **регистрацию ТСД** на сервере WMS — ввести номер терминала.

### Шаг 4: Авторизация на ТСД

в EME БД после регистрации пользователь попадает на экран авторизации. Для авторизации:

1. Открыть диалог **«Складской персонал»** в рабочей станции
2. Выбрать кладовщика
3. Нажать кнопку **«Печать»** в левой части диалога — на экран будет выведен штрих-код бейджа
4. Сканировать сгенерированный штрих-код с экрана компьютера с помощью ТСД
5. Авторизация пройдена успешно
